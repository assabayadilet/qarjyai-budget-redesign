# Icon Sources

This project currently uses `lucide-react` icons as placeholders for fast development.

To switch to your preferred icon packs:

1. Download the final icon set from:
   - https://www.flaticon.com/
   - https://thenounproject.com/
2. Export icons in SVG format.
3. Place SVG files in `src/assets/icons/`.
4. Replace `categoryIcon` and tab icons in `src/App.tsx` with imported SVG components.
5. Add required license attribution text below.
