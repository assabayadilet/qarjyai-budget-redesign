# QarjyAI Budget Tracker Redesign

Standalone Vite + React design prototype for a mobile budget tracker with a liquid glass UI direction.

## Included screens

- Home
- Budget
- Insights
- Goals
- Profile
- Add transaction sheet
- Goal creation sheet
- Goal details sheet

## Stack

- React 19
- TypeScript
- Vite
- Motion

## Local development

```bash
cd "/Users/adiletassabay/Downloads/qarjy ai design"
npm install
npm run dev
```

Default local URL:

```text
http://127.0.0.1:5175/
```

## Build

```bash
npm run build
```
